# St. Kateri Thunder Robotics

A Next.js website for St. Kateri Thunder Robotics Team.

## Getting Started

Install dependencies and run locally:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deploy to Vercel via GitHub

1. Push this folder to a new GitHub repository
2. Go to [vercel.com](https://vercel.com) and click **Add New → Project**
3. Import your GitHub repo
4. Leave all settings as default — Vercel auto-detects Next.js
5. Click **Deploy**

Your site will be live in ~60 seconds.

## Customization

- **Team names/roles**: Edit `pages/index.js` → Teams section
- **Sponsor logos**: Replace placeholder text in the Sponsors section with `<img>` tags
- **Contact email**: Update the `mailto:` link in the CTA section
- **Social links**: Update footer link `href` values
- **Colors**: All color tokens are in `styles/globals.css` under `:root`
